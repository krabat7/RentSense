# Telegram-бот RentSense

Отправка уведомлений о выгодных объявлениях (прогноз модели выше реальной цены) по фильтрам пользователя.

## Команды

- `/start` — регистрация и краткая инструкция
- `/on` — включить уведомления
- `/off` — выключить уведомления
- `/filters` — текущие фильтры
- `/set ключ значение` — установить фильтр; `/set ключ сброс` — сбросить один
- `/reset_filters` — сбросить все фильтры
- `/status` — статус подписки и счётчик уведомлений за день

Фильтры: `district`, `rooms`, `area_min`, `area_max`, `price_min`, `price_max`, `metro`, `travel_time_max`.

## Расписание

Алерты формируются по объявлениям за текущий день, приоритет — по разнице (прогноз − цена). За один запуск каждому пользователю отправляется не более одного объявления. Расписание задаётся через cron (каждый час с 9 до 23) или через `job_queue` при установленном пакете `python-telegram-bot[job-queue]`.

## База данных

Таблицы `bot_users`, `sent_alerts` создаются при первом запуске в `init_bot_tables()`.
