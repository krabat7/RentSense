# Telegram бот RentSense

Бот для отправки уведомлений о новых объявлениях на аренду квартир.

## Функциональность

- Сканирование новых объявлений 1-2 раза в день
- Отправка до 5 уведомлений в день на пользователя
- Фильтрация по предпочтениям пользователя
- Дедупликация уже отправленных объявлений

## Команды

- `/start` - начать работу с ботом
- `/help` - справка по командам
- `/status` - проверить статус подписки

## Настройка

В `.env` файле необходимо указать:

```
TELEGRAM_BOT_TOKEN=your_bot_token_here
TELEGRAM_ALERT_LIMIT=5
TELEGRAM_SCAN_INTERVAL_HOURS=12
```

## Запуск

### Ручной запуск

```bash
python -m app.bot.telegram_bot
```

### Интеграция с планировщиком

Добавить в `app/scheduler/crontab.py`:

```python
from app.bot.scheduler import run_alert_job

# Запуск 2 раза в день (например, в 9:00 и 21:00)
schedule.every().day.at("09:00").do(run_alert_job)
schedule.every().day.at("21:00").do(run_alert_job)
```

## База данных

Бот использует две таблицы:

- `bot_users` - пользователи бота и их предпочтения
- `sent_alerts` - история отправленных алертов для дедупликации

Таблицы создаются автоматически при первом запуске через `init_bot_tables()`.
