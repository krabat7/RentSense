# Рекомендации по прокси для парсинга ЦИАН

## Количество прокси

**Рекомендуемое количество: 10-15 прокси**

**Обоснование:**
- В ocenomet использовалось 10-15 прокси для сбора 80к объявлений за месяц
- Для аренды (rent) данных может быть меньше, чем для продажи (sale), но все равно нужно достаточно прокси
- С текущей логикой парсера:
  - Каждый прокси делает запрос каждые 20 секунд после успешного запроса
  - При блокировке прокси "отдыхает" 5-10 минут
  - С 10-15 прокси можно обрабатывать ~30-45 запросов в минуту
  - Это позволяет собрать 50-100 объявлений в час = 1200-2400 в день = 36k-72k в месяц

**Минимум:** 5-7 прокси (будет работать медленнее)
**Оптимум:** 10-15 прокси
**Максимум:** 20-25 прокси (больше не имеет смысла, т.к. ограничение на стороне ЦИАН)

## Страны прокси

**Рекомендуемые страны (в порядке приоритета):**

1. **Россия** ⭐⭐⭐⭐⭐
   - Лучший выбор для парсинга российского сайта
   - Низкая задержка
   - Меньше подозрений со стороны ЦИАН
   - Рекомендуется: 60-70% прокси из России

2. **Белоруссия** ⭐⭐⭐⭐
   - Хорошая альтернатива России
   - Похожий IP-пул
   - Рекомендуется: 10-15% прокси

3. **Казахстан** ⭐⭐⭐⭐
   - Также хороший выбор для российского сайта
   - Рекомендуется: 10-15% прокси

4. **Украина** ⭐⭐⭐
   - Можно использовать, но может быть больше блокировок
   - Рекомендуется: 5-10% прокси

5. **Другие страны СНГ** ⭐⭐
   - Могут работать, но с большим риском блокировок
   - Рекомендуется: 0-5% прокси

**НЕ рекомендуется:**
- США, Европа, Азия - высокий риск блокировок
- VPN-прокси - часто блокируются
- Публичные бесплатные прокси - нестабильны

## Типы прокси

**Рекомендуется: резидентные (residential) прокси**

1. **Residential прокси** ⭐⭐⭐⭐⭐
   - Выглядят как обычные домашние IP
   - Меньше риск блокировок
   - Дороже, но надежнее
   - Рекомендуется для долгосрочного парсинга

2. **Datacenter прокси** ⭐⭐⭐
   - Дешевле, но чаще блокируются
   - Можно использовать как дополнение
   - Рекомендуется: 20-30% от общего количества

3. **Mobile прокси** ⭐⭐⭐⭐
   - Хорошая альтернатива residential
   - Меньше блокировок
   - Дороже, но эффективнее

## Провайдеры прокси

**Рекомендуемые провайдеры:**

1. **Bright Data (Luminati)** - премиум, дорого, но очень надежно
2. **Smartproxy** - хорошее соотношение цена/качество
3. **Oxylabs** - качественные прокси, средняя цена
4. **Proxy-Cheap** - бюджетный вариант для России
5. **Soax** - хорошие residential прокси

## Настройки парсера

Текущие настройки оптимизированы для защиты прокси:

- **Задержка после успешного запроса:** 20 секунд
- **Задержка при 403/429:** 5 минут
- **Задержка при captcha:** 10 минут
- **Задержка при других ошибках:** 2 минуты
- **Автоматическая разморозка:** каждые 10 минут проверяются заблокированные прокси
- **Параллельная обработка:** до 5 комбинаций room/sort одновременно

## Мониторинг

Рекомендуется:
1. Регулярно проверять логи на блокировки прокси
2. Использовать скрипт `test_proxies.py` для проверки работоспособности
3. Заменять заблокированные прокси новыми
4. Мониторить количество собранных объявлений

## Оценка стоимости

**Примерная стоимость на месяц:**
- Residential прокси (10-15 шт): $50-150/месяц
- Datacenter прокси (10-15 шт): $20-50/месяц
- Смешанный вариант (10 residential + 5 datacenter): $70-100/месяц

**Рекомендация:** Начать с 10-12 residential прокси из России, при необходимости добавить еще 3-5.
